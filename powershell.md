# Useful command line entries for Windows powershell

Display processes
```powershell
Get-Process
```
Rm
```powershell
Remove-Item
Remove-Item -Path <DirectoryPath> -Recurse -Force
```
mkdir
```powershell
New-Item -ItemType Directory <-Path> <nameofthepath>
```
mv
```powershell
Move-Item -Path <SourcePath> -Destination <DestinationPath>
```
